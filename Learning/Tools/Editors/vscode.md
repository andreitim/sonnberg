## Shortcuts

* `code .` -> run vscode with current directory set to the location where the command was run.
- `ctrl + shift + p` -> Command Pallete
- ctrl ` -> Terminal
- `ctrl + ,` -> File / Preferences / Settings 
- `ctrl + p` -> Go to File
- `ctrl + shift +  f` -> Find in Files
- `ctrl + space` -> Show snippets
- Open preview
